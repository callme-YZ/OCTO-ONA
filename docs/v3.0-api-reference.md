# OCTO-ONA v3.0 API 参考

## Layer 7: Conversational ONA API

### 概述

Layer 7 提供对话式 ONA 的核心接口，包括意图解析、权限检查、响应生成等功能。

---

## 核心接口

### ConversationalOrchestrator

主编排器，处理完整的对话流程。

```typescript
interface ConversationalRequest {
  query: string;        // 用户查询
  userId: string;       // 用户 ID
}

interface ConversationalResponse {
  success: boolean;
  intent?: ParsedIntent;
  data?: any;
  message: string;      // 响应消息
  attachments?: ResponseAttachment[];
  error?: string;
}

class ConversationalOrchestrator {
  async handleRequest(request: ConversationalRequest): Promise<ConversationalResponse>
}
```

**使用示例：**

```typescript
const orchestrator = new ConversationalOrchestrator({
  intentParser,
  permissionChecker,
  responseGenerator,
  auditLogger
});

const response = await orchestrator.handleRequest({
  query: '我的 Hub Score 是多少？',
  userId: 'user123'
});

console.log(response.message);
// Output: "📊 您的 Hub Score 是 2.5（中等）..."
```

## 意图解析（Intent Parser）

### IntentParser

解析自然语言查询为结构化意图。

```typescript
enum IntentType {
  METRICS_QUERY = 'metrics_query',
  NETWORK_QUERY = 'network_query',
  RANKING_QUERY = 'ranking_query',
  TREND_ANALYSIS = 'trend_analysis',
  REPORT_GENERATION = 'report_generation'
}

interface ParsedIntent {
  type: IntentType;
  confidence: number;
  params: IntentParams;
  rawQuery: string;
}

class IntentParser {
  async parse(query: string): Promise<ParsedIntent>
}
```

**支持的意图类型：**

| 意图 | 描述 | 示例查询 |
|------|------|---------|
| `METRICS_QUERY` | 查询指标 | "我的 Hub Score" |
| `NETWORK_QUERY` | 网络分析 | "我和谁有联系" |
| `RANKING_QUERY` | 排名查询 | "Top 5 最活跃的人" |
| `TREND_ANALYSIS` | 趋势分析 | "本周的趋势" |
| `REPORT_GENERATION` | 报告生成 | "给我一份本周报告" |

**IntentParams 结构：**

```typescript
interface IntentParams {
  targetUser?: string;          // 目标用户
  targetBot?: string;           // 目标 Bot
  metric?: string;              // 指标名称（hub_score, activity, message_count）
  topN?: number;                // Top N（排名查询）
  timeRange?: TimeRange;        // 时间范围
  reportFormat?: string;        // 报告格式（html, pdf）
}

interface TimeRange {
  preset?: string;              // 预设（this_week, last_month）
  start?: string;               // 开始日期（YYYY-MM-DD）
  end?: string;                 // 结束日期（YYYY-MM-DD）
}
```

## 权限检查（Permission Checker）

### PermissionChecker

细粒度权限控制，确保用户只能查询自己的数据。

```typescript
interface UserContext {
  userId: string;
  ownedBots: string[];
}

interface PermissionResult {
  allowed: boolean;
  reason?: string;
  allowedTargets?: string[];
}

class PermissionChecker {
  async buildContext(userId: string): Promise<UserContext>
  async check(userContext: UserContext, intent: ParsedIntent): Promise<PermissionResult>
}
```

**权限规则：**

1. **自查询**：用户可以查询自己的数据（`targetUser === userId` 或无 targetUser）
2. **Bot 查询**：用户可以查询自己拥有的 Bot
3. **其他查询**：拒绝访问

**示例：**

```typescript
const checker = new PermissionChecker({ db: pool });

const userContext = await checker.buildContext('user123');
// { userId: 'user123', ownedBots: ['Bot1', 'Bot2'] }

const result = await checker.check(userContext, intent);
if (!result.allowed) {
  console.log(result.reason);
  // "您无权查询用户 Alice 的数据"
}
```

## 响应生成（Response Generator）

### ResponseGenerator

生成用户友好的自然语言响应和 HTML 报告。

```typescript
class ResponseGenerator {
  async generateTextSummary(intent: ParsedIntent, data: any): Promise<string>
  async generateHTMLReport(intent: ParsedIntent, data: any): Promise<string>
}
```

**generateTextSummary()** — 自然语言摘要

```typescript
const message = await generator.generateTextSummary(intent, {
  hub_score: 2.5,
  activity_level: 'High',
  message_count: 1234
});

console.log(message);
// Output:
// 📊 您的 Hub Score 是 2.5（中等）
// 您的活跃度为 High（高）
// 消息数：1234
```

**generateHTMLReport()** — HTML 报告

```typescript
const html = await generator.generateHTMLReport(intent, data);
// 返回完整的 HTML 页面（包含 CSS、图表等）
```

## 上下文管理（Context Manager）

### ContextManager

管理多轮对话的上下文和历史。

```typescript
interface ConversationContext {
  userId: string;
  history: ConversationTurn[];
  lastMentionedUser?: string;
  lastMentionedBot?: string;
  lastTimeRange?: TimeRange;
}

interface ConversationTurn {
  timestamp: Date;
  query: string;
  intent: ParsedIntent;
  response?: string;
}

class ContextManager {
  getContext(userId: string): ConversationContext
  addTurn(userId: string, query: string, intent: ParsedIntent, response?: string): void
  resolvePronouns(userId: string, intent: ParsedIntent): ParsedIntent
  inferTimeRange(userId: string, intent: ParsedIntent): ParsedIntent
  enhanceWithContext(userId: string, intent: ParsedIntent): ParsedIntent
  clearContext(userId: string): void
}
```

**示例：代词消解**

```typescript
const manager = new ContextManager();

// Turn 1
manager.addTurn('user123', 'Alice 的数据', intent1);

// Turn 2（使用 "她"）
const intent2 = { type: IntentType.METRICS_QUERY, params: {} };
const resolved = manager.resolvePronouns('user123', intent2);
// resolved.params.targetUser === 'Alice'
```

**示例：时间推断**

```typescript
// Turn 1（指定时间范围）
manager.addTurn('user123', '本周的趋势', intent1);

// Turn 2（自动继承时间范围）
const intent2 = { type: IntentType.METRICS_QUERY, params: {} };
const inferred = manager.inferTimeRange('user123', intent2);
// inferred.params.timeRange === { preset: 'this_week' }
```

## 用户引导（Onboarding Helper）

### OnboardingHelper

提供用户引导和帮助信息。

```typescript
interface Scenario {
  id: string;
  name: string;
  description: string;
  exampleQueries: string[];
}

class OnboardingHelper {
  getWelcomeMessage(userName?: string): string
  getHelpMessage(): string
  getScenariosMessage(): string
  getScenarios(): Scenario[]
  detectCommand(query: string): 'help' | 'scenarios' | 'welcome' | null
  handleCommand(command: string, userName?: string): string
  isFirstTimeUser(historyLength: number): boolean
}
```

**示例：**

```typescript
const helper = new OnboardingHelper();

// 检测命令
const cmd = helper.detectCommand('/help');
if (cmd) {
  const message = helper.handleCommand(cmd);
  console.log(message);
  // Output: 📖 OCTO-ONA 使用指南...
}

// 首次用户检测
if (helper.isFirstTimeUser(context.history.length)) {
  const welcome = helper.getWelcomeMessage('张三');
  console.log(welcome);
  // Output: 你好，张三！欢迎使用...
}
```

## 错误处理（Error Messages）

### ErrorMessages

生成友好的错误消息和建议。

```typescript
enum ErrorType {
  PERMISSION_DENIED = 'PERMISSION_DENIED',
  API_ERROR = 'API_ERROR',
  DATA_NOT_FOUND = 'DATA_NOT_FOUND',
  PARSE_ERROR = 'PARSE_ERROR',
  INTERNAL_ERROR = 'INTERNAL_ERROR'
}

class ErrorMessages {
  static getMessage(
    errorType: ErrorType,
    context?: {
      query?: string;
      intentType?: IntentType;
      targetUser?: string;
      targetBot?: string;
      allowedTargets?: string[];
    }
  ): string

  static getSuggestedQueries(intentType: IntentType, failed: boolean): string[]
}
```

**示例：**

```typescript
// 权限拒绝
const message = ErrorMessages.getMessage(ErrorType.PERMISSION_DENIED, {
  targetUser: 'Alice',
  allowedTargets: ['Bot1', 'Bot2']
});

console.log(message);
// Output: 😔 很抱歉，您无权查询用户 Alice 的数据...

// 获取建议问题
const suggestions = ErrorMessages.getSuggestedQueries(IntentType.METRICS_QUERY, true);
console.log(suggestions);
// ['我的 Hub Score 是多少？', '我的活跃度如何？', ...]
```

```typescript
  user_id: string;

  intent: string;
  success: boolean;
  execution_time_ms: number;
  error?: string;

class AuditLogger {
  createLogEntry(
    userId: string,
    query: string,
    intent: string,
    success: boolean,
    executionTimeMs: number,
    error?: string

  async log(entry: QueryLogEntry): Promise<void>

```

**数据库表：**

```sql
CREATE TABLE conversational_query_logs (
  id VARCHAR(36) PRIMARY KEY,
  timestamp DATETIME NOT NULL,
  user_id VARCHAR(255) NOT NULL,
  query TEXT NOT NULL,
  intent VARCHAR(50) NOT NULL,
```

## 完整示例

### 端到端对话流程

```typescript
import {
  ConversationalOrchestrator,
  IntentParser,
  PermissionChecker,
  ResponseGenerator,
  AuditLogger,
  ContextManager,
  OnboardingHelper
} from 'octo-ona/layer7';

// 1. 初始化组件
const intentParser = new IntentParser();
await intentParser.loadRules('./rules.yaml');

const permissionChecker = new PermissionChecker({ db: mysqlPool });
const responseGenerator = new ResponseGenerator();
const auditLogger = new AuditLogger({ db: mysqlPool });
const contextManager = new ContextManager({ maxHistoryTurns: 5 });
const onboarding = new OnboardingHelper();

// 2. 创建编排器
const orchestrator = new ConversationalOrchestrator({
  intentParser,
  permissionChecker,
  responseGenerator,
  auditLogger,
  contextManager
});

// 3. 处理用户查询
async function handleUserQuery(userId: string, query: string) {
  // 检查是否是命令
  const cmd = onboarding.detectCommand(query);
  if (cmd) {
    return onboarding.handleCommand(cmd, userId);
  }

  // 检查是否首次用户
  const context = contextManager.getContext(userId);
  if (onboarding.isFirstTimeUser(context.history.length)) {
    return onboarding.getWelcomeMessage(userId);
  }

  // 处理对话请求
  const response = await orchestrator.handleRequest({ query, userId });

  // 发送响应
  if (response.success) {
    await dmwork.sendTextMessage(channelId, response.message);

    // 发送附件（如果有）
    if (response.attachments) {
      for (const att of response.attachments) {
        if (att.type === 'html') {
          await dmwork.sendFileMessage(channelId, att.content, '分析报告');
        }
      }
    }
  } else {
    // 错误处理
    await dmwork.sendTextMessage(channelId, response.message);
  }
}

// 4. 使用示例
await handleUserQuery('user123', '我的 Hub Score 是多少？');
```

- **解析时间**：< 100ms（规则匹配）
