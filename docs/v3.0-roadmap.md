# OCTO-ONA v3.0 Roadmap

**Version:** v3.0.0 - Conversational ONA  
**Status:** In Development  
**Target Release:** 2026-04-22

---

## 🎯 Vision

从"BI 工具"到"协作助手"：通过自然语言对话进行组织网络分析。

**核心理念：**
- 🗣️ 用户用自然语言提问，而非操作复杂界面
- 🔒 细粒度权限（只能查自己+自己的Bot）
- 📊 动态生成分析报告
- 🤖 AI 助手而非分析工具

---

## 🎯 MVP Scope

### Supported Questions (5 types)

| Type | Example Question | API Mapping |
|------|-----------------|-------------|
| **Health** | "我最近怎么样？" | HealthMonitor |
| **Trend** | "相比上周有什么变化？" | TrendAnalyzer |
| **Network** | "我和谁协作最多？" | NetworkGraph + Metrics |
| **Bot Activity** | "我的 Bot A 最近做了什么？" | Bot Activity API |
| **Report** | "生成本周报告" | ReportGenerator |

### Permissions

**MVP Rules:**
- ✅ Users can query: self + owned bots
- ❌ Users cannot query: other users, team-level, org-level

### Data Sync

**Strategy:**
- Daily incremental sync @ 3:00 AM
- Manual trigger: `octo-ona sync --now`
- Data freshness: displayed in every response

## 📝 Acceptance Criteria

**v3.0.0 is done when:**

1. ✅ Users can ask questions in DMWork
2. ✅ System understands 5 types of questions (90%+ accuracy)
3. ✅ Permission checks work (deny unauthorized queries)
4. ✅ Responses include text summary + HTML report
5. ✅ Daily auto-sync runs successfully
6. ✅ Response time <5 seconds (90% of queries)
7. ✅ Test coverage >80%
8. ✅ Documentation complete
